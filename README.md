# Inter-Trade_Test_Task

Приложение выполнено в трех слоях:
1. DAL -> Содержит обощенный репозиторий для работы с данными, а также конфиг для создания БД и создания подключений. 
          По умолчанию БД создается в папке Запускающего проекта, сиречь в слое WebApi
2. BL -> Содержит сервисы по работе с данными, модели и ДТО апи.
3. WebApi -> Содержит сервисы по расстановке эндпоинтов, middleware-обработчик исключений и блок di для репозитория и сервисов
             К апи подключен Swagger для удобства работы

Подготовка и запуск: развертывание базы данных происходит при первой попытке обращения к ней, поэтому нужно запустить проект Inter-Trade_Test_Task.WebApi и перейти к Swagger-у
